Course Description

This course provides a broad overview of computer networks. The objective is to focus on an understanding of fundamental concepts of modern computer network architecture (primarily the Internet) from a design perspective. Topics covered include: OSI Architecture, applications protocols such as HTTP, Email and DNS; reliable data transport, routing algorithms, congestion management, and error management. Students will be expected to do systems/network programming and make use of simulation and measurement tools to gain an appreciation of current Internet. Recommended background: CS 2223, CS 2303, C programming, basic probability, and some differential calculus (knowledge of OS concepts covered in CS3013 would be ideal). Note that this course is only the beginning in the study of modern networks. More advanced topics such as wireless networks, sensor networks will be covered in CS 4516.
